### 2️⃣ 🔖 收藏/书签系统

**功能描述：**
- 收藏感兴趣的历史事件
- 右上角收藏面板统一管理
- 本地存储，数据持久化
- 一键删除不需要的收藏

**技术实现：**
```javascript
// 收藏数据存储在 LocalStorage
let bookmarks = JSON.parse(localStorage.getItem('historyBookmarks') || '[]');

// 添加/删除收藏
function toggleBookmark() {
  const index = bookmarks.indexOf(currentEventId);
  if (index > -1) {
    bookmarks.splice(index, 1); // 删除
  } else {
    bookmarks.push(currentEventId); // 添加
  }
  localStorage.setItem('historyBookmarks', JSON.stringify(bookmarks));
}
```

**使用方式：**
1. 点击任意事件打开详情弹窗
2. 点击右上角 ☆ 按钮收藏（变为 ★ 表示已收藏）
3. 点击控制栏"🔖 收藏"按钮查看收藏列表
4. 点击收藏项快速查看详情
5. 点击 × 删除收藏

**快捷键：** `Ctrl+B` 打开/关闭收藏面板

### 4️⃣ ⏱️ 时间缩放过滤

**功能描述：**
- 按时间尺度过滤显示的事件
- 四种过滤模式：全部/千年/世纪/年代
- 快速定位特定时间跨度的事件

**过滤规则：**
| 模式 | 显示范围 | 示例事件 |
|------|---------|---------|
| 全部时间 | 所有事件 | 全部 35+ 事件 |
| 千年尺度 | 1000 年以上 | 人类起源、农业革命 |
| 世纪尺度 | 100-1000 年 | 文艺复兴、黑死病 |
| 年代尺度 | 100 年以内 | 互联网、AI 崛起 |

**技术实现：**
```javascript
function filterByTimeScale(scale) {
  const events = document.querySelectorAll('.event');
  events.forEach(event => {
    const year = Math.abs(parseInt(event.dataset.year) || 0);
    let show = true;
    
    if (scale === 'millennium') {
      show = year >= 1000 || event.dataset.year < -1000;
    } else if (scale === 'century') {
      show = year >= 100 && year < 1000;
    } else if (scale === 'decade') {
      show = year < 100;
    }
    
    event.style.display = show ? 'block' : 'none';
  });
}
```

### 5️⃣ 🔗 相关事件推荐

**功能描述：**
- 智能推荐与当前事件相关的其他事件
- 基于共同标签进行匹配
- 在事件详情弹窗底部展示
- 最多推荐 3 个相关事件
- 点击直接跳转查看详情

**推荐算法：**
```javascript
const relatedEvents = historicalEvents
  .filter(e => e.id !== eventId && 
               e.tags.some(t => event.tags.includes(t)))
  .slice(0, 3);
```

**示例：**
- 查看"互联网诞生" → 推荐"个人电脑普及"、"万维网发明"、"人工智能崛起"
- 查看"古埃及金字塔" → 推荐"苏美尔文明"、"罗马帝国"、"古希腊民主"
