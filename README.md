# 🌍 人类历史发展进程

Timeline of Human History - From Origins to Present

## 🌐 在线访问

**https://magicengine-ai.github.io/human-history/**

## 📋 项目说明

这是一个交互式的人类历史时间线网站，展示从人类起源到现代的重大历史事件。

### ✨ 主要特点

- 📅 **5 大历史时期**：史前时期、古代文明、中世纪、近代早期、现代
- 🎯 **50+ 重大事件**：人类起源、农业革命、文字发明、工业革命、世界大战、登月、互联网、AI 时代等
- 🔍 **搜索功能**：快速查找历史事件、文明、人物
- 🏷️ **事件分类**：文化、技术、政治、科学四大类别
- 💬 **详细信息**：点击任意事件查看深度解读
- 📱 **响应式设计**：支持手机、平板、桌面设备
- 🎨 **现代 UI**：深色主题、流畅动画、进度条

## 📖 历史时期划分

| 时期 | 时间范围 | 主要特征 |
|------|----------|----------|
| 🪨 史前时期 | 300 万年前 - 公元前 3500 年 | 人类起源、火的控制、农业革命 |
| 🏛️ 古代文明 | 公元前 3500 年 - 476 年 | 文字发明、金字塔、罗马帝国 |
| 🏰 中世纪 | 476 年 - 1453 年 | 封建制度、宗教、蒙古帝国 |
| ⛵ 近代早期 | 1453 年 - 1789 年 | 文艺复兴、大航海、科学革命 |
| 🚀 现代 | 1789 年 - 至今 | 工业革命、世界大战、信息时代 |

## 🎯 功能特点

- **时间线导航**：垂直时间线展示，清晰的历史脉络
- **事件筛选**：按时代筛选或查看全部
- **全文搜索**：支持关键词搜索历史事件
- **分类标签**：每个事件标注文化/技术/政治/科学类别
- **详情弹窗**：点击事件查看详细的背景、影响和意义
- **滚动进度**：顶部进度条显示浏览进度
- **交互动画**：悬停、点击波纹等流畅动画效果

## 🎨 设计理念

- **沉浸式体验**：深色宇宙主题，象征历史的深邃
- **金色时间线**：象征人类文明的珍贵历程
- **分类色彩**：不同类别使用不同颜色，便于识别
- **渐进式披露**：先概览后详情，避免信息过载
